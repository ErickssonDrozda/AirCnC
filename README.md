# Omnistack Week - AirCnC

## Languages of Development
- Frontend: ReactJs
- Backend: NodeJs
- Mobile: React Native
- Database: Nosql Mongodb

## What's this ? 
AirCnC is a platform for companies to offer places, earn money and still have the chance to look for some talent
